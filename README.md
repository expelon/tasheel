# Tasheel Consultancy Website

A modern React-based website for Tasheel Consultancy, providing business setup and consultancy services.

## Features

- Responsive design built with React and TypeScript
- Styled with Tailwind CSS
- Modern UI with smooth animations
- Multiple sections: Home, Company, Services, Gallery, Contact
- Optimized for Vercel deployment

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
3. Open http://localhost:3000 in your browser

## Build for Production

```bash
npm run build
```

## Deploy to Vercel

The project is configured for Vercel deployment. Simply connect your repository to Vercel and it will automatically deploy.

## Technologies Used

- React 19
- TypeScript
- Tailwind CSS
- Vite
- Lucide React Icons
- Motion (Framer Motion)
